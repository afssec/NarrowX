How Auto Extract Works

  1. Auto-scroll the page to trigger lazy-loaded assets
  2. Enable navigation guard to block window.open and unwanted navigations
  3. Perform safe synthetic clicks on selected elements (no new tabs)
  4. Wait until the network becomes idle
  5. Mine endpoints from performance entries and monitored XHR/Fetch
  6. Parse inline scripts and external JS via background (CSP/CORS-safe)
  7. Scan attributes (href, src, action)
  8. Apply Scope filter + exclude non-API resources (css/img/...)
